Ques. Difference between div and span

In HTML DIV and SPAN are tags they belongs to non-semantics category , DIV is an block level element block elements takes the entire width and they always start from the new line , SPAN is an inline element they take the width as much is required by the content , they do not start from the new line.

Ques. Difference between class and IDs

In HTML Class and ID are the identifier that are used to identify the elements and also used to tag these elements in the embedded style sheets with thier specific properties , most important difference between Classes and Id are classes can be used by more than one element in a page , while ID is only used for only single element . In External Style Sheets Classes are represented by '.' symbol and ID are represented by '#' symbol.

Ques. What are self closing tags

A self-closing tag in HTML is a kind of HTML tag that does not need to be closed manually by its closing tag, which means it does not have a separate closing tag as . Some few self-closing tags are input, hr, br, img, etc.
